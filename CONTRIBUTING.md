# Contributing to Jialu Guidebooks

Thank you for your interest in contributing! Here's how you can help.

## 🌍 Translation Improvements

Every guide exists in both Chinese and English. If you spot translation issues:

1. Open an Issue describing the problem
2. Or submit a PR with the fix — please edit only the relevant language file

## ✏️ Typos & Formatting

- Fork the repo
- Fix the issue in the relevant `zh-CN/` or `en/` file
- Submit a Pull Request with a clear description

## 💡 Topic Suggestions

Want a guide on a specific AI topic? [Open an Issue](https://github.com/chengjialu8888/Jialu-Guidebooks/issues/new) with:
- The topic you'd like covered
- Why you think it would be valuable
- Any specific aspects you want addressed

## 📏 Style Guide

When contributing content, please follow these conventions:
- Use clear, practical language — avoid unnecessary jargon
- Include code examples and screenshots where helpful
- Keep the bilingual structure: every change in `zh-CN/` should have a corresponding update in `en/` (or flag it for translation)

## 🚫 What We Don't Accept

- AI-generated content without human review and editing
- Promotional content or affiliate links
- Content that doesn't align with the "builder-first" philosophy
